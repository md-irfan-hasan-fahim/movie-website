### Run Locally

To run **movie** locally, run this command on your git bash:

Linux and macOS:

```bash
sudo git clone https://github.com/md-irfan-hasan-fahim/movie-website.git
```

Windows:

```bash
git clone https://github.com/md-irfan-hasan-fahim/movie-website.git
```
